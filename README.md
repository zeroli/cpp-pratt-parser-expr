## Introduction
This is an expression parser that uses pratt-parser algorithm.
It supports below prefix, infix and postfix operators:
- prefix: `*`, `&`, `+`, `-`, `++`, `--`, `()`
- infix: `+`, `-`, `*`, `/`, `==`, `!=`, `<`, `>`, `<=`, `>=`, `&&`, `||`, `=`, `?:`
- postfix: `[]`, `++`, `--`
